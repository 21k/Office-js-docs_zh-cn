## 常见问题解答

### 如何获取 GitHub 帐户？

填写[加入 GitHub](https://github.com/join) 中的表格，以开立免费 GitHub 帐户。 

### 从何处获取参与者许可协议？ 

拉取请求请求获取此协议时，将会自动向你发送需要签署参与者许可协议 (CLA) 的通知。 

社区成员**必须签署参与许可协议 (CLA) 才能向项目提供大量提交内容**。只需完成并提交此文档一次。请仔细查看该文档。可能要求你让你的员工签署此文档。

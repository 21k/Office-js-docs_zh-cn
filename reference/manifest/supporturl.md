
# SupportUrl 元素
指定提供外接程序支持信息的页面的 URL。

 **外接程序类型：**内容、任务窗格、邮件


## 语法：


```XML
<SupportUrl DefaultValue="string " />
```


## 可以包含：

[Override](../../reference/manifest/override.md)


## 属性



|**属性**|**类型**|**必需**|**说明**|
|:-----|:-----|:-----|:-----|
|DefaultValue|URL|必需|指定该设置的默认值，表示为 [DefaultLocale](../../reference/manifest/defaultlocale.md) 元素中指定的区域设置。|

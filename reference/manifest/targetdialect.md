
# TargetDialect 元素
 定义该词典支持的、表示为区域性名称字符串的方言。

 **外接程序类型：**任务窗格


## 语法：


```XML
<TargetDialect>
   string 
</TargetDialect>
```


## 包含在：

[TargetDialects](../../reference/manifest/targetdialects.md)


## 注解

指定值采用 BCP 47 语言标记格式，如 `"en-US"`。
